## Get Started
---

```shell
$ git clone https://github.com/ryullulala/Brewfile.git ~/Brewfile
$ brew bundle --file=~/Brewfile/private.Brewfile
```

## If you want to make Brewfile
---

```shell
$ brew bundle dump --file=~/Brewfile/private.Brewfile
```
